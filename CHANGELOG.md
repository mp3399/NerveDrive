# Changelog

All notable changes to this project are documented here. Format: [Keep a Changelog](https://keepachangelog.com/); versioning: [SemVer](https://semver.org/).

## [0.1.0] - 2026-07-04
### Added
- Initial release: in-browser Apple Health `Export.zip` parsing via streaming Web Worker.
- Full analysis engine (activity, cardio, sleep, body, workouts, metabolic, correlations, anomalies, population, scores, coach plan).
- Interactive ECharts dashboard with dark/light themes.
- Export to PDF, Markdown, CSV and JSON.
- Accessibility, responsive layout, and privacy-first architecture (no backend).
